# Gym Chatbot Prompt Lab

Basic console chatbot for a gym. It answers simple questions about gym services, hours, and general info.

## Requirements

- Node.js 18+ (works with newer versions too)
- npm

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

Type your message in the terminal. Use `exit` to quit.

## Notes

- Prompts live in `src/prompts/`.
- Main entry point: `src/index.ts`.
